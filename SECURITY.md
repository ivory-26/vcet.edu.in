# Security Policy

## Supported Versions

Currently, only the latest version on the `main` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

Please **DO NOT** report security vulnerabilities via public GitHub issues, as that could potentially put the website or the users at risk.

Instead, please send an email to vcet_inbox@vcet.edu.in or open a private vulnerability report on GitHub if available for this repository.
We will respond as quickly as possible to coordinate a patch and a security advisory.
